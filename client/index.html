pixelwarz
A try at making a simple multiplayer game with an authorative server.
Move with WASD keys, attack with Space key, and abilities at 1,2,3 keys.
Source code at: github.com/daemoned/pixelwarz
The code is sort of a mess as I have been learning both JavaScript and TypeScript while doing the project.
Players
Spectators

Connect - Prompts for a name and opens a WebSocket connection so you get realtime updates and can chat.
Play - Join the gameplay if there is a spot open.
Spectate - Realtime updates and chat but no participation in gameplay.
Disconnect - Closes the WebSocket connection.
